# UNPACKEDFOOD DB

## CREAZIONE DOCKER
```bash
docker run -d --name mysql -p 0.0.0.0:3306:3306 -e MYSQL_ROOT_PASSWORD=*** --restart=always mysql:8.0.20
```

## ACCEDERE AL DATABASE
```bash
docker exec -it mysql mysql -u root -p
```

## CREAZIONE DATABASE
```sql
CREATE DATABASE UnPackedDB;
```

## CREAZIONE TABELLA
```sql
CREATE TABLE UNPACKEDFOOD (
    Unique_ID INT NOT NULL PRIMARY KEY,
    Name VARCHAR(255) NOT NULL,
    Average_Expiry_Days INT NOT NULL,
    Category VARCHAR(50) NOT NULL
);
```
